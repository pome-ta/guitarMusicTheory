# guitarMusicTheory

ギターで音楽理論をやる 😤


## はじめに

ギターと音楽理論を同時に習得するのは難しみあると思う。なので、双方の考え方を分割し習得を目指す。



私の復習もあるので、ゴチャゴチャてんやわんやしそうだけどNE！

### 想定している手順

1. ギターのコード・フォーム(ボイシング) 
1. コードの響きの理解
1. 鍵盤(楽譜)上解説と、ギターコードとの紐付け
1. そして伝説へ
